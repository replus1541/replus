리플러스에 대한 소개

저는 UI / UX 디자인 실무자이며 미래의 웹 환경을 연구하는 사람입니다

리플러스에서는 디자이너들도 알기쉽게 업계 트렌드와 기술 관련 정보를 정리해 공유하고있습니다.

초심자를 위한 UI 기본상식

UI 디자인을 공부하는 초심자들은, 대부분 시각적인 부분에만 집중하는 경우가 많습니다. 그렇기에 리플러스 채널에서는 웹 생태계와 기본 환경. 그리고 실제 UI 디자인의 기본 유닛 단위에 대한 내용을 자료화하여 알려드리고있습니다.

설계와 역기획

대부분의, UI 디자이너들은 설계와 역기획의 중요성에 대해서 잘 알지 못하는 경우가 많습니다. 그래서 리플러스 채널에서는 개별 서비스에 대한 기술스택 분석과, 설계적인 장단점, 메뉴구조에 대한 UI 요소까지 분석하여 이야기를 드리고 있습니다.

웹개발과 프레임워크

UI 디자이너의 시대는 갔고, 이제 개발과 프레임워크의 시대가 왔습니다. 리플러스 채널에서는 부트스트랩에서부터 시작해 자바스크립트 프레임워크에 이르기까지. 디자이너도 알기 쉽게 웹 프레임워크를 다룰 수 있는 방법에 대해 이야기하고있습니다.

해외문서 번역 / 영상 컨텐츠

대부분의 UI 디자인 관련 컨텐츠들은 국내보다 해외의 관련 소식을 알아두는것이 중요합니다. 그렇기에 리플러스에서는 자체적인 번역 컨텐츠를 업로드하고, 중요한 자료는 영상을 제작하여 안내를 드리고있습니다

리플러스의 채널안내

현재 운영하고있는 채널은
브런치, 오픈 카톡방, 유튜브 채널입니다

현재는 총 세가지 채널이 운영되고있지만, 추후에는 별도의 채널이 생성될수도 있습니다.

리플러스가 하는일

개발자와 UI 디자이너 사이에서
고민하는 분들을 돕고있습니다

UI디자인에 대한 체계적인 정보정리를 통해 웹개발 관련 지식을 공유하고있습니다.

IT 마켓 트렌드

수없이 많은 기술들이 새로 생겨나고있는 IT업계에서는 새로운 소식들이 계속해서 등장합니다. 그렇기에 새로운 웹과 UI 기술과 업계의 변화, 커리어에 대한 트렌드를 정리해 알려드립니다.

실전 웹개발 프로젝트

UI 디자인을 넘어서 웹 개발과 자바스크립트 프레임워크까지. 일반적인 UI 디자이너도 개발을 공부할 수 있다는걸 직접 증명하기 위해 지속적인 개발 관련 연구를 진행하고있습니다. 또한 이 과정에서 모든 연구내용의 자료화를 진행하고있습니다.

실전 그로스해킹

UI 디자인은 이제 웹개발과 사업의 영역의 단계로 들어섰습니다. 그렇기에 기획과 사업적인 역량. 채널과 서비스를 성장시키는 체계적인 방법이 중요해졌습니다. 리플러스는 이런 실무 그로스해킹을 진행하면서, 직접 겪은 그로스해킹에 대한 자료화를 진행하고있습니다.

모바일과 웹, OS 환경에 대한 실험

UI 디자이너들은 대부분 모바일과 PC 웹에 대한 규격만을 고민하는 경우가 많습니다. 그러나 IT 생태계에는 다양한 기기들과 UI 규격들이 존재합니다. 그렇기에 리플러스는 개별 OS와 주변기기들에 대한 연구와 분석도 진행하고있습니다.

영상 컨텐츠와 오프라인 강의

사람은 누구나 자신이 알고있는 것들을 머리속에 갖고있지만. 가르치는 과정을 통해서 더 많은것들을 배우게됩니다. 리플러스는 영상 컨텐츠나, 오프라인 강의를 통해서 많은 분들께 지식의 공유정신을 실천해나가고있습니다.

1:1 문의와 개별상담

많은 사람들의 고민을 한데 모아 정리하고, 실무자들이 어떤 어려움을 겪는지를 체감하는건 리더로서 굉장히 중요한 경험입니다. 그렇기에 리플러스는 온라인상으로 1:1 문의나, 개별 상담을 진행하고있습니다.
